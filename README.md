# Photo Studio

ブラウザベースのLightroom風 写真管理・現像ツール（MVP）

## 機能

- **ライブラリ管理** — サムネイルグリッド、レーティング（★0〜5）、採用/不採用フラグ、フィルタリング
- **非破壊編集** — 露出、コントラスト、ハイライト、シャドウ、白/黒レベル、色温度、色被り、彩度、自然な彩度、明瞭度、かすみの除去
- **リアルタイムプレビュー** — Canvas 2D によるピクセル単位の画像処理
- **ヒストグラム** — RGB + 輝度のリアルタイム表示
- **Before/After比較** — `\` キーで切替
- **自動補正** — ヒストグラム分析に基づく自動調整
- **エクスポート** — フル解像度 JPEG 書き出し

## キーボードショートカット

| キー | 機能 |
|------|------|
| `0-5` | レーティング |
| `P` | 採用フラグ |
| `X` | 不採用フラグ |
| `←→` | 写真送り |
| `Enter` | 現像モードへ |
| `Esc` | ライブラリへ戻る |
| `\` | Before/After |
| `R` | 調整リセット |
| ダブルクリック | スライダー個別リセット |

## セットアップ

```bash
npm install
npm run dev
```

## 技術スタック

- React 18 + Vite
- HTML5 Canvas 2D（画像処理エンジン）
- インメモリストレージ（MVP）
